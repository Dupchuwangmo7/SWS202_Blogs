### **What is the Bash Prompt?**

- The **Bash prompt** is the **text-based indicator** in the terminal that shows the system is ready for input.
- It usually includes:
  - **Username** (`<username>`)
  - **Hostname** (`<hostname>`)
  - **Current working directory** (`<cwd>`)

### **Default Prompt Format**

```bash
<username>@<hostname>:<current_directory>$
